# Banking Management System (Core Java)

## Overview

A console-based Banking Management System built using Core Java to simulate real-world banking operations while strengthening object-oriented programming fundamentals.

This project was developed without using collection frameworks to reinforce mastery of core Java concepts such as arrays, inheritance, polymorphism, encapsulation, and method overriding.

---

## Features

* Create Savings Account
* Create Current Account
* Deposit Funds
* Withdraw Funds
* Search Account by Account Number
* Transfer Funds Between Accounts
* Display All Accounts

---

## Business Rules Implemented

### Savings Account

* Minimum balance of **1000** must be maintained after withdrawal

### Current Account

* Supports overdraft up to **5000**
